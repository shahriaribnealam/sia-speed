# ⚡ SIA Speed - Internet Speed Test Tool

**SIA Speed** is a modern, web-based internet speed testing application designed with a stunning glassmorphism user interface. It simulates real-time network performance analytics including **Download Speed, Upload Speed, Ping, and Jitter** across highly optimized global edge servers.

This project is fully optimized for performance, accessibility, and search engine visibility (SEO).

## 🚀 Live Demo
Experience the live application here: **[[sia-speed.pages.dev](https://sia-speed.pages.dev/)](https://sia-speed.pages.dev)**

---

## ✨ Features

- **Global Server Selection:** Simulate tests using top-tier data centers worldwide (Bangladesh, India, Singapore, UAE, Germany, UK, USA, Australia).
- **Realistic Network Algorithms:** Network parameters (latencies, pings, and bandwidth caps) dynamically adjust based on geographical distances from South Asia.
- **Fluid Gauge Animation:** Real-time feedback with a beautiful, custom SVG circular speedometer gauge.
- **Premium UI/UX:** Built with a modern Dark Mode Glassmorphism layout (`backdrop-filter`) that looks premium on any display.
- **100% Mobile Optimized:** Fully responsive layout with custom-centered interface controls, preventing UI overflows on mobile viewports.
- **SEO & Social Share Ready:** Pre-configured with clean meta tags, Open Graph (OG), and Twitter Cards for rich link previews.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Flexbox/Grid, Custom CSS Transitions)
- **Icons & Typography:** Font Awesome 6.4.0, Google Fonts (Poppins)
- **Core Logic:** Vanilla JavaScript (Asynchronous Tasks, Promises, Dynamic DOM manipulation)
- **Deployment:** Hosted on Cloudflare Pages for ultra-fast edge delivery.
